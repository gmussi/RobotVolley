## Long description

Robot Volley is a fast, floaty arcade volleyball game set in a futuristic neon stadium. Two customizable robot athletes rally a glowing ball over the net — first to **5 points** wins.

**Features:**
- **1 Player** vs smart CPU or **2 Player** local multiplayer
- **Robot Lab** — mix and match heads, torsos, arms, and legs with unique abilities
- **Part lottery** — random upgrades between rallies keep every match fresh
- **Neon sci-fi arena** with synth-pop soundtrack
- Play in browser or install on mobile via the web app

Classic arcade volleyball feel — position and timing control the ball, not power spam. Hold serve to charge, smash with the Hand arm above the speed cap, and deflect with axe and ninja star throws.
